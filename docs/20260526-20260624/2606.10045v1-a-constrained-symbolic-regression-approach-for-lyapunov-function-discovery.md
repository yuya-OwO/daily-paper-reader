## 摘要
本文考虑自治动力系统的李雅普诺夫函数的数据驱动发现。我们将李雅普诺夫函数表示为固定深度的表达式树，并将李雅普诺夫发现任务表述为有约束的自监督符号回归问题。约束条件对给定输入的李雅普诺夫函数输出以及李雅普诺夫稳定性条件进行建模。这种建模方法不对李雅普诺夫函数的函数形式做任何先验假设，由于以符号形式获得函数，因此本质上是可解释的，并且原则上可应用于任何连续动力系统。我们还开发了一种定制的分支-定界-检查求解方法，以有效解决由此产生的学习任务。多个案例研究的应用显示了所提方法发现李雅普诺夫函数的能力。

## Abstract
In this paper, we consider the data-driven discovery of Lyapunov functions for autonomous dynamical systems. We represent the Lyapunov function as an expression tree of fixed depth and formulate the Lyapunov discovery task as a constrained self-supervised symbolic regression problem. The constraints model the output of the Lyapunov function for a given input as well as the Lyapunov stability conditions. This modeling approach makes no a priori assumptions about the functional form of the Lyapunov function, is inherently interpretable since the function is obtained in a symbolic form, and, in principle, can be applied to any continuous dynamical system. We also develop a tailored branch-and-bound-and-check solution approach to efficiently solve the resulting learning task. Applications to several case studies show the ability of the proposed approach to discover Lyapunov functions.

## 2. 论文提出的方法论
### 2.1 核心思想
将李雅普诺夫函数建模为**固定深度的完美二叉树（表达式树）**，通过选择每个节点上的运算符/操作数（+、−、×、÷、exp、log、常数、状态变量）来参数化候选函数。这构成了一个**混合整数非线性规划（MINLP）**问题，其中二进制变量决定节点操作，连续变量决定常数及中间节点值。

### 2.2 关键约束
- **树定义约束**：保证运算符的结对正确（如二元运算符必须有两个子节点）、状态变量必须全部出现等。
- **值定义约束**：通过大M法将节点值与子节点值按所选运算符关联（如加法：\(v_{n}=v_{2n}+v_{2n+1}\)）。
- **李雅普诺夫约束**：
  - 正定性：对非原点数据点 \(V(x_{ij}) > 0\)，对原点 \(V(0)=0\)；
  - 导数负定性：用前向差分近似 \(\dot V\)，即 \(\frac{V(x_{i,j+1}) - V(x_{ij})}{\Delta t} \le -\alpha \|x_{ij}\|^2\)，\(\alpha\) 为收敛率。

### 2.3 目标函数
可组合优化项：
- 最大化收敛率（\(-\alpha\)）；
- 最小化函数复杂度（运算符总数）；
- 正则化常数大小（\(\sum c_n^2\)）。

### 2.4 求解算法：分支-定界-检查
- 采用标准空间分支定界（spatial branch-and-bound）求解MINLP。
- **创新点**：在求解过程中，每当找到一个整数可行解（即一个候选函数表达式），立即调用**验证子程序**：
  - 计算 \(\min_{x\in \mathcal{D}} V(x)\) 和 \(\max_{x\in \mathcal{D}} \dot V(x)\)（通过全局优化）；
  - 检查 \(V(0)=0\)；
  - 若三个条件均满足，则立即终止搜索，返回该有效李雅普诺夫函数；
  - 否则继续分支定界。
- 若分支定界完成仍未找到有效函数，则提示增加数据或调整参数。
