## 摘要
本文解决了离线学习一个输出反馈替代控制器的问题，该替代控制器近似给定的、可能计算昂贵的非线性控制器-观测器对。替代模型被建模为递归动态系统，并通过模仿给定控制器生成的闭环输入/输出轨迹进行训练。除了模仿精度，离线训练问题通过纳入估计状态轨迹来学习候选李雅普诺夫函数，从而促进输入到状态实际稳定性。该方法在非线性连续搅拌釜式反应器上进行了验证，通过概率验证方法评估了约束满足和实际稳定性。数值结果通过与仅模仿的基线进行比较，突出了联合学习李雅普诺夫函数的优势。

## Abstract
This paper addresses the problem of learning an output-feedback surrogate controller offline that approximates a given, possibly computationally expensive, nonlinear controller-observer pair. The surrogate is modeled as a recurrent dynamical system and is trained to imitate closed-loop input/output trajectories generated by the given controller. Beyond imitation accuracy, the offline training problem promotes input-to-state practical stability by incorporating estimated state trajectories to learn a candidate Lyapunov function. The approach is validated on a nonlinear continuous stirred tank reactor, where constraint satisfaction and practical stability are assessed through a probabilistic validation approach. The numerical results highlight the benefit of jointly learning the Lyapunov function by comparing against an imitation-only baseline.

## 2. 论文提出的方法论
### 核心思想
- 将替代控制器建模为具有内部状态的循环动态系统，其输入为历史输出测量值和参考值，输出为控制动作。
- 训练目标包括三部分：
  1. **模仿损失**：最小化替代控制器输出与专家控制器输出的均方误差。
  2. **李雅普诺夫正则化**：通过约束两步非单调李雅普诺夫下降条件，鼓励闭环系统的实际稳定性。
  3. **平衡点一致性正则化**：确保替代控制器在稳态时满足自洽条件。
- 证明在满足两步李雅普诺夫下降条件并考虑测量噪声时，闭环系统具有输入到状态实际稳定性（ISpS）。

### 关键技术细节
- **替代控制器结构**：由递归内部状态更新函数 \(z_{k+1}=f_z(z_k, y_k, \bar{y};\theta)\) 和控制输出函数 \(\hat{u}_k = f_u(z_k, y_k, \bar{y};\theta)\) 构成，其中 \(f_z\) 采用带tanh激活的神经网络 + 仿射部分，\(f_u\) 通过裁剪保证控制约束。
- **李雅普诺夫函数设计**：选取二次型加正定项的形式 \(V(\tilde{\xi}) = \|\tilde{\xi}\|_P^2 + \mu_V \|\tilde{\xi}\|_2^2\)，其中 \(P = \psi_V^\top \psi_V\) 保证半正定。
- **两步下降条件**：要求 \(\tau V(\tilde{\xi}_{k+2}) + V(\tilde{\xi}_{k+1}) - (1+\tau)V(\tilde{\xi}_k) \leq -S(\tilde{\xi}_k)\)，允许能量在一步内上升但两步内总体下降。
- **训练算法**：通过Adam + L-BFGS-B联合优化所有参数，损失函数为：
  \[
  \min_\Theta\; J(\mathcal{D}_t;\theta) + J_L(\mathcal{D}_t;\psi,\mu,\tau) + J_s(z_s) + r_\Theta(\Theta)
  \]
  其中 \(J_L\) 基于估计状态近似计算李雅普诺夫下降的违规惩罚。
- **ISpS定理**：在满足一定假设条件下，若存在函数使得 \(\Gamma_k \leq \alpha_\eta(\|(\eta_{k+1},\eta_k)\|) + c\)，则闭环系统ISpS。

- **理论保障**：证明了两步非单调李雅普诺夫条件在噪声下导出ISpS，并提供了概率验证框架，增强了可信度。

## 8. 不足与局限
- **实验覆盖**：仅在一个CSTR案例上验证，未涉及更多非线性系统（如机器人、化工过程等），通用性有待进一步检验。
- **偏差风险**：李雅普诺夫正则化基于估计状态而非真实状态，可能引入近似误差；若估计状态偏差较大，可能导致李雅普诺夫条件不可靠。
- **应用限制**：
  - 需要离线获得专家控制器的完整轨迹，适用于可预采集数据的场景。
  - 假设马尔可夫性（当前输出和内部状态），对于非最小相位系统或长时滞系统可能效果不佳。
  - 李雅普诺夫函数形式固定为二次型，对高度非线性系统的表达能力有限。
- **计算资源**：训练虽然仅需CPU，但需优化多个正则项，超参数敏感，调参成本较高。
- **理论局限**：ISpS证明依赖于函数 Lipschitz 连续性等假设，实际系统可能不严格满足。
