## 摘要
我们提出了一种直接数据驱动的方法，用于由未知线性时不变子系统组成的无限网络的控制器综合。利用从每个子系统收集的一组噪声污染输入-状态轨迹，并在某些线性矩阵不等式成立的前提下，通过局部构造指数输入-状态稳定（eISS）控制Lyapunov函数以及指数输入-状态稳定反馈控制器，使每个子系统达到指数输入-状态稳定。然后，我们在无限维空间中将这些局部组件在组合小增益条件下组合，得到全局控制Lyapunov函数和相关的稳定控制器，确保无限网络的均匀全局指数稳定性。该方法在一个具有未知动力学的物理案例研究中得到验证。

## Abstract
We propose a direct data-driven method for controller synthesis of infinite networks composed of unknown linear time-invariant subsystems. Using a single set of noise-corrupted input-state trajectories collected from each subsystem, and provided that certain linear matrix inequalities hold, each subsystem is rendered exponentially input-to-state stable (eISS) by locally constructing an eISS control Lyapunov function together with an exponentially input-to-state stabilizing feedback controller. We then compose these local components under a compositional small-gain condition in infinite-dimensional spaces to obtain a global control Lyapunov function and an associated stabilizing controller, ensuring uniform global exponential stability of the infinite network. The approach is validated on a physical case study with unknown dynamics.

### 2. 方法论：核心思想、关键技术细节
- **核心思想**：将无限网络分解为未知LTI子系统，基于每个子系统采集的**单组含噪输入-状态轨迹**，通过求解**线性矩阵不等式（LMI）** 局部构造**指数ISS（eISS）控制Lyapunov函数**及**指数输入-状态稳定反馈控制器**；然后在无限维空间中验证**组合小增益条件**，将局部组件合成为**全局控制Lyapunov函数**和**稳定控制器**，保证网络均匀全局指数稳定（UGES）。
- **关键技术细节**：
  - **子系统模型**：每个子系统为连续时间LTI：\(\dot{x}_i = A_i x_i + B_i u_i + D_i w_i\)，其中\(A_i, B_i\)未知，\(D_i\)已知（表示互联权重），\(w_i\)由邻居状态组成。
  - **数据采集**：对每个子系统采集N个样本，记录输入矩阵\(U_i\)、状态矩阵\(X_i\)，并通过邻居状态构造\(W_i\)。状态导数通过数值差分获得，并考虑有界噪声（已知上界\(\Psi_i\)）。
  - **局部eISS设计**：假设存在二次型eISS控制Lyapunov函数\(V_i(x_i)=x_i^\top P_i x_i\)和反馈控制律\(u_i = K_i P_i x_i\)。通过LMI（公式12）求解\(P_i, K_i\)，该LMI融合了数据一致性约束（S-procedure）和噪声界，确保\(V_i\)满足eISS不等式：\(\dot{V}_i \le -\kappa_i V_i + \rho_i |w_i|^2\)。
  - **组合小增益条件**：定义增益矩阵\(\Phi = K^{-1}\Delta\)，其中\(\Delta\)的元素\(\delta_{ij} = \rho_i / \alpha_j\)，\(\kappa_i\)为局部衰减率。要求谱半径\(r(\Phi) < 1\)，且参数一致有界（公式14-15）。该条件下，可构造全局Lyapunov函数\(V(x) = \sum_i \eta_i V_i(x_i)\)（\(\eta_i\)为正权重），并得到全局衰减率\(\kappa_\infty\)，从而证明闭环网络的UGES。
  - **混合网络**：局部证书（LMI）可源于数据驱动或已知模型，因此适用于部分已知、部分未知的混合网络。
