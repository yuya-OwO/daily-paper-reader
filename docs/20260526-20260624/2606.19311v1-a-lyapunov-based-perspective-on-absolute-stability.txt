# A Lyapunov-Based Perspective on Absolute Stability 

## Tessina H. Scholl ∗

Abstract 

This article presents a unifying perspective on absolute stability concepts. In particular, it develops a Lyapunov-like explanatory framework for a nonscalar circle criterion with its small-gain and strict-passivity special cases. To this end, a general defining inequality for a Lyapunov-like function is proposed that avoids strict definiteness conditions, enabled by a strengthening of the sector con-straint. We discuss different ways to derive a quadratic solution: via a linear matrix inequality (LMI), an alge-braic Riccati equation, and a matrix equation. By exploit-ing the Kalman–Yakubovich–Popov (KYP) lemma, clas-sical frequency-domain results are recovered. A passivity-index-based result is derived that simplifies the evalua-tion. Overall, the presented interrelations may be useful for both analysis and teaching. 

Keywords: absolute stability, Lyapunov, robustness, non-linear systems, sector constraints, KYP lemma 

# 1 Introduction 

The main objective of this article is to present a unify-ing perspective on absolute stability results, emphasiz-ing the interrelations between various concepts. To this end, a simple and transparent Lyapunov-like explanatory framework for a nonscalar circle criterion with its small-gain and strict-passivity special cases is developed. Ex-planatory frameworks for absolute stability criteria are nowadays predominantly Plancherel–Parseval-based ap-proaches, which typically arise from an input-output-operator setting [1, 2, 3]. However, Lyapunov-based ap-proaches are also possible [4, 5, 6, 7, 8, 9, 10, 11]. Histori-cally, both even emerged in parallel; see, e.g., [8, 12] for ex-tensive literature reviews. Plancherel–Parseval-based ap-proaches including the concept of integral quadratic con-straints (IQCs) [1] are more far-reaching, but Lyapunov-based approaches offer a key advantage: regional stability results are still obtained if the derived sector constraints are only met locally [4]. We refer to the presented result as a Lyapunov-like func-tion since the approach in this article does not insist on strict definiteness properties, which are, e.g., encountered in [4, Thm. 7.1], [6, Thm. 1], [7, Thm. 2.2], [8, eq. 11], and [9, p. 29]. # 2 Problem description 

We consider a nonlinear, possibly uncertain, system ˙x(t) = 

f (t, x (t)) . Well-posedness is addressed by Assumption 2.3 below. Assuming that f (t, 0n) = 0 n for all t ∈ R, we are interested in properties of the zero equilibrium. The main result of this article concerns attractivity of this equilib-rium and, in particular, the regional or global character of this property. In addition, a result on uniform asymptotic stability will also be derived. 

▶ Definition 2.1 (Attractivity). Let x(t) = x(t; t0, x 0) de-note a maximal solution with initial value x(t0) = x0 ∈ Rn

of a system with an equilibrium at x = 0 n. The zero equi-librium is locally attractive for a given initial time t0 ∈ R

if there exists a set S ⊆ Rn with 0n ∈ int( S) such that 

∀x0 ∈ S : ∥x(t; t0, x 0)∥ → 0 as t → ∞ , (1) and globally attractive if S = Rn. ◁

As a first step, we decompose the system into 

˙x(t) = Ax (t) + g(t, x (t)) , (2) with a nominal linear part ˙x = Ax , A ∈ Rn×n, and a possi-bly nonlinear term g : [ t0, ∞)×Rn → Rn that is henceforth referred to as perturbation . For instance, 

• g(t, x ) might be the remainder of a time-invariant lin-earization (otherwise differentiability is not assumed) 

• g(t, x ) might involve a saturation term or related non-linearities, or 

• g(t, x ) = ∆( t)x, ∆( t) ∈ Rn×n might address additive uncertainties in the system matrix A.

The construction principle for V

The guiding idea is as follows. We seek a function V for which the derivative is less than or equal to −w, because then the derivative is at most zero, wherever w is at least zero—and the latter characterizes the sector. Some comments 

Concerning the construction of V according to (18): Afunction V that satisfies (18) can easily be found, provided the chosen sector-describing quadratic form w(ζ, α ) from Sec. 3 is compatible with the system’s robustness. Corre-sponding approaches will be discussed in Sec. 6. In fact, these approaches do not only establish (18) for a special function a(t, ζ ) of interest but (18) will hold simultane-ously for arbitrary a(t, ζ ).

▶ Remark 4.1 (Relation to dissipativity). Once (18) holds for any u ∈ Rm in place of −a(t, ζ ), i.e., D+(A · +Bu )V (x) ≤−w(Cx, −u), the latter can be interpreted as a QSR-dissipativity property of (A, B, C ) [5]. Thus, as an alterna-tive approach to the one in this article, it can be examined whether the combination (4) with a sector-constrained nonlinearity is a neutral interconnection of dissipative el-ements [5, 6]. ◁

Concerning the set Ω0 from (19): Note that Ω0 only depends on the sector defined by w and on the nonlin-earity a. It describes where the nonlinearity resides in the sector from Sec. 3 in terms of the points x ∈ Rn for which α = a(t, ζ ) with ζ = Cx satisfies the sector inequal-ity w(ζ, α ) ≥ 0. If the graph of the nonlinear function 

ζ 7 → a(t, ζ ) is globally within the sector (for any t), then 

Ω0 = Rn. (21) If the sector is chosen such that at least for small values of

# 12 Conclusion 

The presented approach is based on a general defining inequality for the Lyapunov-like function V . Using a quadratic ansatz, a corresponding function V can be com-puted via an LMI, an ARE, or a matrix equation. The ap-proach relies on an inequality chain that ultimately leads to the sector-defining inequality. The introduction of the offset function ℓ, which can be chosen as a general class-K

function—thus giving rise to the concept of a K-reduced sector—is a key distinction from existing approaches to ab-solute stability. Moreover, a proof that relies on Barbalat’s lemma combined with the BIBS and 0-CICS property of a stabilized system is presented. For the considered system class, it enables Lyapunov-like functions that may lack strict definiteness requirements. A passivity-index formu-lation is derived for the circle criterion. Furthermore, the article clarifies key interrelations between various concepts in the field of absolute stability.
