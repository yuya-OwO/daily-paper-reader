## 摘要
我们建议使用李雅普诺夫特征指数（LCE）作为强化学习问题中稳定垂直运动倒立摆的稠密奖励信号。借助LCE，智能体不仅成功找到了称为卡皮查摆的振荡运动，还使摆的枢轴阻尼，将其保持在严格竖直位置。

## Abstract
We suggest using the Lyapunov characteristic exponent (LCE) as a dense reward signal for the reinforcement learning problem of stabilizing the inverted pendulum with vertical motion. With LCE, the agent not only successfully found the oscillatory motion known as the Kapitza pendulum but also damped the pendulum's pivoting, leaving it in a strictly upright position.

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：将最大Lyapunov指数（LCE）作为强化学习的奖励信号。LCE是衡量动力系统相空间中相邻轨迹指数发散速率的指标，负值表示系统稳定。通过在线计算LCE，智能体能直接获得系统是否趋于稳定的反馈，从而避免稀疏奖励问题。
- **关键技术细节**：
    - **环境建模**：采用无量纲化的倒立摆运动方程：\(\ddot{\varphi} = (1 - a)\sin\varphi\)，其中\(a(t)\)是竖直升加速度（动作）。状态包括\(\varphi, \dot{\varphi}\)以及枢轴位置\(y_c\)等。
    - **LCE计算**：遵循Benettin等人方法，将运动方程与切空间方程\(\ddot{\delta} = (1 - a)\delta\cos\varphi\)联立求解。每0.25秒（50个时间步）计算一次最大LCE：\(\nu_1 = \lim_{k\to\infty} \frac{1}{k s}\sum_{i=1}^k \ln\alpha_i\)，其中\(\alpha_i\)为每步归一化后的切向量范数。
    - **奖励函数**：\(r_t = -\nu_1 - 100\frac{(y_c/y_{\text{max}})^{20}}{1+(y_c/y_{\text{max}})^{20}}\)。第一项驱使LCE负向（稳定），第二项惩罚枢轴离开中心区域（\(y_{\text{max}}=10\) cm），超出边界则终止并惩罚-1000。
    - **状态表示**：每个时间步的状态为过去50步的\(y_c, \dot{y}_c, \sin\varphi, \cos\varphi, \dot{\varphi}\)组成的250维向量。
    - **学习算法**：Soft Actor-Critic (SAC)，使用默认超参数（学习率1e-4，折扣0.99，软更新率0.005，批大小256等），两层256神经元全连接网络。
